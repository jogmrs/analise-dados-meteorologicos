# Análise de Dados Meteorológicos

# Descrição
Projeto acadêmico desenvolvido em Python para processamento e análise de dados meteorológicos.

# Objetivo
Calcular médias de temperatura e precipitação com base em dados históricos.

# Aplicação
- Armazenamento de dados meteorológicos
- Inserção de novas medições
- Cálculo de médias dos últimos 12 meses
  
# Resultado
O sistema permite analisar dados climáticos e gerar médias anuais de forma automatizada.
